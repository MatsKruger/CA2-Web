Hobbysøgning

Søg efter hobbyer. Returnerer de hobbyer som opfylder kriteriet.

GET http://dawa.aws.dk/hobbies

Parametre

Navn Beskrivelse
id Hobbyens unikke id, f.eks. 1.

Eksempler

Find hobby med id 1

http://localhost:8080/CA2/api/hobbies?id=1

Hobbyopslag

Modtag hobby med id.

GET http://dawa.aws.dk/hobbies/{id}

Parametre

Navn Beskrivelse
id Hobbyens unikke id, f.eks. 1.

Eksempler

Returner hobby med id 1

http://localhost:8080/CA2/api/hobbies/1

Beskrivelse af hobbydata

Web API'et udstiller for hver hobby følgende informationer:

Navn Type Beskrivelse
href string Hobbyens unikke URL.
id int Hobbyens id
name string Hobbyens navn
description string Hobbyens beskrivelse
