Personsøgning

Søg efter personer. Returnerer de personer som opfylder kriteriet.

GET http://dawa.aws.dk/persons

Parametre

Navn Beskrivelse
id Personens unikke id, f.eks. 1.
hobby Id på den hobby som du vil have personer fra

Eksempler

Find person med id 1

http://localhost:8080/CA2/api/persons?id=1

Find alle personer med hobby id 1

http://localhost:8080/CA2/api/persons?hobby=1

Personopslag

Modtag person med id.

GET http://dawa.aws.dk/persons/{id}

Parametre

Navn Beskrivelse
id Personens unikke id, f.eks. 1.

Eksempler

Returner person med id 1

http://localhost:8080/CA2/api/persons/1

Beskrivelse af persondata

Web API'et udstiller for hver person følgende informationer:

Navn Type Beskrivelse
href string Personens unikke URL.
id int Personens id
firstName string Personens fornavn
firstName string Personens efternavn
email string Personens email
phones array Personens telefoner
id integer Telefonens id
number string Telefonens nummer
description string Telefonens beskrivelse
hobbies array Hobbier som array [{id, name, description}...]
id integer Hobbiens id
name string Hobbiens navn
description string Hobbiens beskrivelse
address object Personens adresse
id integer Adressens id
street string Adressens vejnavn
additionalInfo string Adressens husnr, etage og side/dør
cityInfo object Adressens tilhørende by
id integer Byens id
zipCode string Byens postnummer
city string Byens navn
