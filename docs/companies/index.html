Firmasøgning

Søg efter firmaer. Returnerer de firmaer som opfylder kriteriet.

GET http://localhost:8080/CA2/api/companies

Parametre

Navn Beskrivelse
id Firmaets unikke id, f.eks. 1.

Eksempler

Find company med id 1

http://localhost:8080/CA2/api/companies?id=1

Firmaopslag

Modtag firma med id.

GET http://dawa.aws.dk/companies/{id}

Parametre

Navn Beskrivelse
id Firmaets unikke id, f.eks. 1.

Eksempler

Returner firma med id 1

http://localhost:8080/CA2/api/companies/1

Beskrivelse af firmadata

Web API'et udstiller for hvert firma følgende informationer:

Navn Type Beskrivelse
href string Firmaets unikke URL.
id integer Firmaets id
name string Firmaets navn
description string Firmaets beskrivelse
cvr string Firmaets CVR
numOfEmployees integer Firmaets antal af medarbejdere
marketValue string Firmaets markedsværdi
email string Firmaets email
phones array Firmaets telefoner
id integer Telefonens id
number string Telefonens nummer
description string Telefonens beskrivelse
address object Firmaets adresse
id integer Adressens id
street string Adressens vejnavn
additionalInfo string Adressens husnr, etage og side/dør
cityInfo object Adressens tilhørende by
id integer Byens id
zipCode string Byens postnummer
city string Byens navn
